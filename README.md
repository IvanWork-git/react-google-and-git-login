# React Social Authentication Client

![React Social Auth](https://i.imgur.com/acA83LR.gif)


## Getting Started

```
git clone https://github.com/IvanWork-git/react-google-and-git-login.git
cd react-auth-client
npm i && HTTPS=true npm start
```
